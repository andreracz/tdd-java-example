# Requisitos para o exemplo

Implemente um micro serviço REST que calcule os números da sequencia de Fibonacci.

A sequencia de Fibonacci é tal que:

* F(n) = F(n-1) + F(n-2) sendo que:
* F(0) = 0 e F(1) = 1
